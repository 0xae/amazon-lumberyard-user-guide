# Creating a New Level<a name="level-create"></a>

The first step in creating a game world is to create a level\.

**To create a new level**

1. In Lumberyard Editor, click **File**, **New**\.

1. In the **New Level** window, enter a file name and select a directory location for the file\.

1. Select the desired **Heightmap Resolution** and **Meters per Texel** values\. Click **OK**\.

1. In the **Generate Terrain Texture** window, for **Texture Dimensions**, select texture dimensions to determine texture tile resolution\.
